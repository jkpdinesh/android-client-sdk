Provides for different Join meeting request results Success: When join meeting api request is successful
InvalidMeetingIDOrPasscode: When the meeting id or passcode given is wrong
NetworkError: Internet is not connected or bandwidth is very low to make a network call
MeetingInProgress: Meeting is already joined LoginRequired: When not logged in or when forcing moderator login is enabled for meeting
MeetingLocked: Moderator has locked meeting and no more participants are allowed to join
HostAccountDisabled: Account is disabled
MaxParticipantsExceeded: Meeting has reached maximum participants allowed
TryAgainLater: Server has detected too many requests or request is termed unsafe
FraudDetected: Request from unauthorized account or joining secured meeting without access or passcode expired
InternalError: Unable to process the request to an internal error
InsufficientPermissions: Required min permissions are not granted
